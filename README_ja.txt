Still Image Video Renderer
v0.2.0

概要
----
静止画像、音源ファイル（MP3/WAV）、任意のSRT字幕をffmpegで合成し、MP4を作成するPython/Tkinterアプリのプロトタイプです。

主な仕様
--------
- 画像 / 音源 / SRTを個別選択できます。SRTは任意です。
- tkinterdnd2 がある場合、ファイルドロップに対応します。
- 音源と同名のSRTが同じフォルダにある場合は自動設定します。
- SRTがない場合は、静止画と音源だけで字幕なしMP4を作成します。
- 音源だけをドロップ/選択した場合、同じフォルダの「音源名_title.png」も自動検索して画像に設定します。
- 出力ファイル名の自動候補は音源ファイル名.mp4です。既存ファイルがある場合は _001 などを付けます。
- 4x5のクイックパネルを2ページ持ちます。
- クイックパネルは左側に配置し、基本的にパネルへドロップして作業します。
- 1ページ目には通常用とショート用の主要プリセットを並べ、2ページ目は初期状態では空きです。
- パネルにファイルをドロップすると、そのパネルのプリセットを適用し、素材を読み込みます。
- クイックパネルにマウスオーバーするとプリセットのフルネームを表示します。
- パネル適用時はffmpegで1フレームのレンダープレビューを生成します。
- フォント名はドロップダウン（手入力も可）で選択できます。
- 手作業でフォント名、フォントサイズ、Alignment、MarginVを編集できます。
- Alignmentは `6（中央上）` と `10（中央）` から選択します。
- 手作業編集中は簡易プレビューを表示します。
- 「レンダープレビュー更新」でffmpeg実レンダーのプレビューを更新できます。
- 画像未指定時も白い16:9背景でプレビューできます。SRT未指定時はプレビュー用字幕テキストを使います。
- English UIでは、SRT未指定時の既定プレビュー字幕は「Subtitle text for preview」になります。
- 主要UIは日本語/Englishを切り替えできます。
- 「MP4作成」を押すと、その時点の入力と字幕設定をジョブとして固定し、ffmpeg実行中でも次の作業を続けられます。
- 現在設定を新規プリセットとして保存できます。
- 選択中プリセットへの上書き保存もできます。
- 選択中プリセットを消去できます。消去するとクイックパネル上の該当スロットも空きになります。
- 「MP4作成」はクイックパネル下の動画作成セクションから実行します。

注意
----
- 簡易プレビューは近似表示です。最終確認はレンダープレビューを使ってください。
- フォント未指定プリセットでは、簡易プレビューは代理フォントを使います。実出力ではfontname/fontsizeをffmpegへ渡しません。
- 簡易プレビューのフォントサイズ補正係数の初期値は 0.76 です。
- プリセット名とクイックパネル表示名は日本語/English別に保持し、未設定側はもう一方の言語へフォールバックします。
- v0相当のプリセットは、MarginV=0ではなくMarginV未指定として扱います。
- 初期版では、出力解像度は画像サイズそのままです。ffmpeg側で幅・高さを偶数化します。
- 初期版では、画像のcontain/cover/stretch等のフィット処理はありません。

開発実行
--------
run_dev.bat を実行してください。

EXE化
-----
build_exe_onedir.bat を実行してください。
PyInstallerでtkinterdnd2を含めるため、--collect-data tkinterdnd2 を指定しています。
Windowsではffmpeg/ffprobe起動時に黒いコンソール窓が出ないよう、subprocessにはCREATE_NO_WINDOW相当の設定を適用しています。
ffmpeg.exe / ffprobe.exe は bin フォルダに置くと、ビルド後の dist にコピーされます。

ffmpeg探索順
------------
1. アプリフォルダ/bin/ffmpeg.exe, ffprobe.exe
2. アプリフォルダ直下
3. PATH

GitHub用ソースZIPについて
-------------------------
GitHub用ソースZIPには、Pythonソース、プリセット初期値、ドキュメント、バッチファイルを含めます。
ffmpeg.exe / ffprobe.exe、.venv、build、dist、ローカル生成されたpresets.jsonやプレビュー画像は含めません。
