# Codex作業開始用クイックメモ

まず `CODEX_HANDOFF.md` を読んでください。
次に `SPECIFICATION_ja.md` と `CHANGELOG.md` を確認してください。

推奨作業順：

1. `run_dev.bat` で起動確認。
2. `bin/ffmpeg.exe` と `bin/ffprobe.exe` を置く。
3. 音源のみドロップで `<音源名>.srt` と `<音源名>_title.png` が読まれることを確認。
4. パネルドロップ時にレンダープレビューが出ることを確認。
5. GUI改善に着手。

最優先の改善領域：

- パネルスロット編集UI。
- 右ペインのレイアウト整理。
- 全プリセット検索UI。
- 簡易プレビューとレンダープレビューの状態表示改善。
- EXE化時の動作確認。
