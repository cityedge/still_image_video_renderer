# Changelog

## 1.0.0 - 2026-07-09

- 実戦利用版としてバージョンを `1.0.0` に更新。
- 最新のユーザープリセット配置・表示名を `presets_default.json` に反映。
- SRTファイル読み込み時に、同名音源と `<SRT名>_title.png` を自動検出するよう変更。
- SRT起点の自動検出で、前回作業の音源・画像が残っている場合も同名素材へ置き換えるよう修正。
- クイックパネル編集モードを追加。モーダル内でスロットの移動、入れ替え、表示名編集、空き化ができるよう変更。
- クイックパネル編集モードで、表示名適用後に選択状態を解除し、表示名入力欄のEnterキーで適用できるよう変更。
- Alignmentの選択肢に `2（中央下）` を追加。
- 入力ファイル名と出力ファイル名の表示を末尾優先にし、`short` や `_001` などのpostfixを確認しやすく変更。

## 0.2.0 - 2026-06-30

- プリセット名とクイックパネル表示名を `ja` / `en` の言語別フィールドに変更。現在の言語側が未設定の場合はもう一方へフォールバックする仕様に変更。
- バージョンを `0.2.0` に更新。
- エクスプローラーからのドロップ操作を優先し、クイックパネルと動画作成セクションを左ペイン、入力・字幕設定を右ペインへ配置。
- 汎用ドロップエリアを入力ファイル欄の下へ移動し、高さを拡大。
- 全プリセット選択をプリセット操作ボタンの直上へ移動。
- プリセット操作ボタンを「プリセット」付きの2行表示にして横1行に配置。
- 出力ファイル名の行を入力ファイル欄から動画作成セクションへ移動し、ボタン文言を「指定」に変更。
- SRT未指定でも、静止画と音源だけで字幕なしMP4を作成できるよう変更。
- English UI でSRT未指定時のプレビュー字幕既定文言を `Subtitle text for preview` に変更。
- `USER_GUIDE.md` を追加し、ffmpeg / ffprobe へのパラメータ反映を含む利用手順を整理。
- `LICENSE` を追加。
- コンボボックスの選択済み表示をライト背景・黒文字に変更し、言語選択ボックスをコンパクト化。
- 選択中プリセットを消去するボタンを追加し、消去時にクイックパネル上の参照も削除するよう変更。
- `MP4作成` ボタンをクイックパネル下の動画作成セクションへ移動。
- アプリ名を `Still Image Video Renderer` に変更。
- 簡易プレビューのフォントサイズをASS/libassの `PlayResY=288` 基準から画像高へスケールするよう修正。
- 簡易プレビューの字幕位置をテキストブロックの実バウンディングボックス基準で配置するよう修正。
- 簡易プレビューの暫定倍率 `preview_font_scale` を `0.76` に変更。
- `MP4作成` をジョブ化し、ffmpeg実行中でも次の素材選択・調整・MP4作成を続けられるよう変更。
- MP4ジョブ投入時に入力ファイル、出力先、字幕スタイルをスナップショットし、実行中ジョブが以後のUI変更に影響されないよう修正。
- Windowsでffmpeg/ffprobe起動時に黒いコンソール窓が出ないよう、subprocessの共通非表示設定を `run` / `Popen` 両対応に変更。
- PyInstaller onedirビルド用バッチを `.venv\Scripts\python.exe` 直指定にし、配布用ファイルコピーを補強。
- 下部のMP4ジョブ一覧とログを左右2ペイン配置に変更。
- クイックパネルと動画作成セクションを左、入力・字幕設定ペインを右に配置。
- 主要UIラベルを日本語/Englishの言語切替式に変更。
- 画像未指定時も白い16:9背景で簡易プレビュー/レンダープレビューを更新できるよう変更。
- 字幕サイズの `-5`, `-1`, `+1`, `+5` 調整ボタンを追加。
- Alignmentを `6（中央上）` / `10（中央）` のコンボボックスに変更。
- クイックパネル選択時に全プリセットドロップダウンも同期するよう変更。
- クイックパネルにプリセットフルネームのツールチップを追加。
- 初期クイックパネル配置を変更し、1ページ目に通常用とショート用を並べ、2ページ目を空きに変更。

## 0.1.1-prototype

- 出力ファイル名候補を画像名ベースから音源名ベースへ変更。
- 出力ファイル重複時に `_001` 形式で候補を作るよう変更。
- フォント名入力をコンボボックス化。候補外のffmpegフォント名も手入力可能。
- 簡易プレビュー補正係数を `0.5` から `1.0` へ変更。
- 簡易プレビューのMarginV概算を `MarginV × 画像高さ / 288` に変更。
- 音源のみドロップ時に `<音源名>_title.png` を自動検出する機能を追加。
- 既存の `presets.json` から `preview_font_scale=1.0` へ移行する簡易マイグレーションを追加。
